# Plan: v0.2b — Email Markdown Cleanup

## Context

Cached school newsletter emails are bloated (24KB) because `markdownify` faithfully converts HTML layout `<table>` elements into massive markdown tables. The actual content is ~11KB of clean readable markdown. This wastes tokens when Claude reads the cached files.

The fix: add an HTML preprocessing step in `gmail_helpers.py` that unwraps layout tables and strips tracking images **before** markdownify runs. Also shorten tracking URLs to domain-only to preserve link text (important for action items) while cutting URL bloat.

## Design decision: apply to all emails, not just school

Cleanup runs in `body_to_markdown()` for every email, not just school newsletters. This is safe because:
- Plain-text emails skip HTML processing entirely (early return)
- Only HTML-heavy newsletter-style emails have layout tables — regular HTML emails don't
- URL shortening preserves link text and domain; `mailto:` links kept as-is
- Tracking images (no alt text) are never useful regardless of email type

## Changes

### `gmail_helpers.py` — add `_clean_newsletter_html(html)` helper

Called from `body_to_markdown()` before `markdownify.markdownify()`.

Steps:
1. Use `BeautifulSoup` (already a transitive dependency of markdownify) to parse the HTML
2. **Unwrap layout table tags**: `soup.find_all(['table', 'thead', 'tbody', 'tr', 'td', 'th'])` → `tag.unwrap()` — keeps inner content, removes table structure
3. **Remove tracking images**: `soup.find_all('img')` where `alt` is empty → `img.decompose()`
4. Return `str(soup)`

Then in `body_to_markdown()`, after `_strip_cid_and_data_images(html)`, call `_clean_newsletter_html()` before passing to markdownify.

### `gmail_helpers.py` — add `_shorten_tracking_urls(markdown)` helper

Called from `body_to_markdown()` on the markdownify output.

- Regex over markdown links: `[text](url)` → truncate URL to domain + `/...`
- Preserve `mailto:` links as-is
- Collapse 3+ consecutive blank lines to 2

### `gmail_helpers.py` — update `body_to_markdown()`

```python
def body_to_markdown(plain: str, html: str) -> str:
    if plain and not _is_only_comments(plain):
        return plain
    if not html or _is_only_comments(html):
        return ""
    cleaned = _strip_cid_and_data_images(html)
    cleaned = _clean_newsletter_html(cleaned)          # NEW
    md = markdownify.markdownify(cleaned).strip()
    return _shorten_tracking_urls(md)                   # NEW
```
